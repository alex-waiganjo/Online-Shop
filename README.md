# Fully featured Online-Shop

Components:
 - Online store
    - Django(Frontend + Backend), Postgres, Docker
    - Celery(Async Email Sending), Flower, RabbitMQ(Broker)
    - Redis(Recommendation Engine)
    - Payment Gateway(Mpesa)
    - Coupon/discounts feature
    - Multi-language support
 - ETL Pipeline
    - Airflow
    - Dbt
 - Data Visualization
    - Power BI
    - SQL
